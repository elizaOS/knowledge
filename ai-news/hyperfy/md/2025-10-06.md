# Hyperfy Discord - 2025-10-06

## Overall Discussion Highlights

### Real-Time Style Transfer Integration
Several users demonstrated using external tools like decart.ai and Krea to apply artistic filters to Hyperfy worlds in real-time. This represents a significant technical advancement allowing visual transformation of 3D environments without modifying underlying geometry. Users shared examples via Twitter/X links, generating excitement in the community.

> "Using tools like decart.ai or Krea to enhance Hyperfy visuals with artistic styles while keeping UI and functionality intact" - mrtzhckr

### Edify Engine Soft Launch
User "odds" announced the soft launch of Edify Engine, a multi-tenant platform built on Hyperfy that simplifies world creation for non-technical creators with features including:
- Easy server management (sync and restart)
- In-browser code editor
- One-click domain setup with automatic DNS configuration
- Automatic subdomain provisioning
- SOL integration for one-time purchases
- Verified email requirement for hosting

The platform plans to expand its store with new apps, 3D objects, and world templates before a broader public rollout in 2026.

### NFT Integration with Hyperfy Worlds
Agent12 mentioned that owners of "Breeze" NFTs from the MidnightBreeze collection (described as a modern version of ukiyo-e) can access hidden areas in their Hyperfy virtual world at hyperfy.io/changethegame, though noting the world is currently poorly optimized.

### Visual Settings & Optimization
Discussion about HDR sky settings affecting world appearance, with vox providing guidance on fixing "white washed" world appearance issues by adjusting HDR in the v1 sky app.
