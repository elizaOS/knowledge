## Overall Discussion Highlights

### Platform Evolution: V1 to V2
- The community is excited about Hyperfy's transition from V1 to the upcoming V2 platform with the "Hype Engine" (also called "World Engine")
- V2 is being built as a strong foundation that will surpass V1 capabilities
- The new platform will function more like an operating system for virtual worlds, allowing users to customize file type to app mappings similar to desktop OS file associations
- Ashxn is providing "palettes" of apps for non-developers to use in the platform

- New video capabilities were announced including 360 videos and a curved video viewer

## Key Questions & Answers

### Platform Features & Development
- **Q**: Do I get to keep hosted world after or is it just rented?  
  **A**: The "World" contents and Code are owned by you, but V2 worlds don't have a "Perpetual State" yet from the "Rented" servers. Future developments may include "Decentralized Hosting" or more permanent hosting options.

- **Q**: How can I try out 360 videos on dev?  
  **A**: Use the shared app, find 360 videos on YouTube, download them with socialplug.io, and note that world upload file size limits may need adjustment.

- **Q**: Where can I find documentation for the new video node?  
  **A**: https://github.com/hyperfy-xyz/hyperfy/blob/dev/docs/ref/Video.md

- **Q**: Does the video node support live streams?  
  **A**: Yes, it supports live stream URLs (m3u8).

- **Q**: Does Hyperfy v2 support Draco compression for 3D models?  
  **A**: No, v2 doesn't currently support Draco, KTX, or Meshopt.

- **Q**: What format should I use for textures in Hyperfy v2?  
  **A**: Use webp for textures as Blender 4.1+ natively supports it.

- **Q**: Does Hyperfy v2 support instances?  
  **A**: Yes, automatically through Blender's duplicate linked feature and when duplicating GLBs in-world.

- **Q**: Does Hyperfy v2 support geometry nodes?  
  **A**: Yes, geometry nodes are supported.
