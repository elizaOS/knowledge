## Overall Discussion Highlights

### Development & Technical Progress
- **Hyp-Summer Excitement**: Community members expressed enthusiasm about "Hyp-Summer," described as a period where "everyone is leveling up" with many developers actively working on new features.

- **Token Distribution**: Ashxn provided context about token distribution, noting that approximately 10% consists of unclaimed airdrops, 15% is linear vested over 3 years, and 30% is currently inactive DAO allocation.

## Key Questions & Answers

### Development
- **Q**: Can geometry nodes be exported to GLTF?  
  **A**: You can export geonodes but there are limits.

- **Q**: How to fix issues with deleting models?  
  **A**: Try selecting all code with ctrl+a, then ctrl+/ or remove code and try deleting again; if still having trouble, it could be network-related.

- **Q**: What is Hyp-Summer?  
  **A**: "It's everyone leveling up" with many community members "cookin."

- **Q**: Are Hyperfy v1 events still being approved and viewed?  
  **A**: Yes, notification issues were fixed and pending events have been accepted.

### Token Economics
- **Q**: What happens to unclaimed airdrops?

- **Q**: How is the $HYPER token vested?  
  **A**: Another 15% is linear vested over 3 years.
