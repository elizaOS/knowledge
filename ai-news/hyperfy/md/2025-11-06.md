# Hyperfy Discord - 2025-11-06

## Overall Discussion Highlights

### Gaming & Blockchain Applications
- **3rdEyeVisuals** is developing a memecoin-related gambling app, starting with a slot machine
- **Ash** referenced "Fallen" (https://fallen.hyperfy.xyz/), a PvP game where players stake HYPER tokens
- Game mechanics discussed include token staking, player battles, and prize withdrawals
- **3rdEyeVisuals** plans to create a "Wipeout" version with a prize pool, preferring PvE over PvP
- **Pete** mentioned interest in developing a GTA-style game with guns

### Hyperscape & Platform Relationships
- **Hyperscape** was identified as a fork of Hyperfy v2
- **Ash** clarified that Hyperscape uses Hyperfy's core technology but removes features like in-world live-building and scripting
- Discussion about the ethics of naming a fork with a similar name
- **Agent12** claimed money and scammers are moving to Hyperscape, allegedly built by Shaw

### Metaverse Conceptual Discussion
- **Jayzen** proposed that Hyperfy needs a global map to create a true metaverse experience
- This would allow users to visualize and navigate between worlds, creating network effects
- **3rdEyeVisuals** suggested implementing this as an app that detects portal links between worlds
- **mrtzhckr** countered that the metaverse won't be singular but will consist of multiple metaverses for different industries

### Development Workflows
- **3rdEyeVisuals** shared their process for bundling .hyp files for iterative testing
- **Ash** mentioned they simply build in-world and download when needed to share
