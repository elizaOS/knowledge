## Key Questions & Answers

### App Development
- **Q**: How do I edit existing .hyp apps? (asked by SWISS)  
  **A**: You can't edit .hyp files directly; drag them in-world, use the in-world editor to change code, hit save, and wait for the server to save the recompiled hyp to the database (answered by hiroP)

- **Q**: What's the recommended workflow for editing apps? (asked by SWISS)  
  **A**: Drag a .hyp into the world, open the in-world editor, change code, hit save, and wait for the server cycle to compile the app (answered by hiroP)

- **Q**: Do I need to wait for changes to take effect? (asked by Shiffty)  
  **A**: Code changes affect what you're editing instantly, but for apps stored on the server that you want to use later from the menu, you need to wait for the server save cycle (answered by hiroP)
