# Hyperfy Discord - 2025-10-18

## Overall Discussion Highlights

### Video & Media Handling
- Users discussed limitations with uploading large video files to the platform, with disconnection issues occurring for files over 800MB
- A 200MB file at 480p resolution was successfully uploaded for a 10-minute music video
- Performance differences were noted between m3u8 files hosted on GitHub (laggy in v1) versus MP4 files in v2
- Streaming options like Vimeo Pro were recommended as alternatives to direct uploads

### Platform Development
- Token gating functionality from V1 is awaiting reintegration in V2
- Community members shared resources for new users, including example worlds and tutorial videos
- Some discussion about detecting user movement in virtual worlds using Location/Bounding Box functionality

### HYPER Token
- Significant discussion about the HYPER token's price performance, which has apparently dropped by 99%+ since launch
- Community members expressed mixed sentiments - some maintaining conviction in the project fundamentals despite price action, others expressing frustration about lack of communication
- Questions about token utility and future roadmap were raised
