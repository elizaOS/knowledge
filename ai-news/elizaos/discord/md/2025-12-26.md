## Key Questions & Answers

**Q: Why is ElizaOS token price continuously dropping?**  
A: Multiple factors including team management issues, whale manipulation, and lack of X (Twitter) presence for marketing.

**Q: What happened with the token migration from AI16Z to ElizaOS?**  
A: The migration used a snapshot approach which split liquidity and caused confusion, especially for exchange users.

**Q: How is ElizaOS Cloud being used?**  
A: Developers are building AI agents on it, with examples like a marijuana-themed agent and other projects being shared.

**Q: Is the ElizaOS project dead?**  
A: No, development continues with ElizaOS Cloud gaining users despite token price issues.

**Q: What's the difference between the project's technical merits and token value?**  
A: The token can be a bad investment while the technology/project remains valuable.

**Q: Is it possible to upload multiple MD files for an agent?**  
A: Multiple files should work, and file upload optimization is currently in progress.

**Q: Is one of the goals of Jeju for the agents to be truly decentralized and unstoppable?**  
A: Yes, decentralized hosting and agents are very much at the heart of the project.

**Q: Can I plug in an AI agent to help me edit videos in DaVinci Resolve Studio?**  
A: Not directly yet; suggested workaround is to use Gemini's screen share capability for guidance.

**Q: Is it possible to use npm instead of bun?**  
A: No, users need to continue using bun.
