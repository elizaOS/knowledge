# elizaOS Discord - 2025-11-06

## Overall Discussion Highlights

### Token Migration
- The migration from AI16Z tokens to ElizaOS tokens was scheduled to begin but experienced delays
- Several exchanges (MEXC, Binance Alpha, Gate, and Bybit) are supporting the migration with different timelines
- MEXC appears to be the first exchange to complete the swap for users
- The migration process involves minimal gas fees and will remain open for 90 days
- Tokens will be immediately tradeable after migration
- Community members repeatedly asked about migration procedures for self-custody wallets

### Development Updates
- Shaw submitted and R0am merged PR #5 to the otc-agent repository
- Development of a "login with cloud" SDK is underway to enable seamless authentication across multiple applications
- Plans to add an "Apps" section to Cloud where users can whitelist domains and link external apps
- Discussion about plugin-sql dependencies and requests to release the plugin-elizacloud plugin
- Sentry errors were dismissed as a non-issue from a single user

### Community & Promotion
- Coordination for promotional activities including a town hall featuring "Eliza agents"
- Plans to stream promotion for an OpenSea collection called "Art Decc0s"
- Discussion about Discord bot integration with ElizaOS
- Suggestions to improve project communication strategy and marketing

## Key Questions & Answers

### Token Migration
- **Q**: How do I swap AI16Z tokens for ElizaOS on exchanges?  
  **A**: Different exchanges have their own processes; check exchange announcements

- **Q**: Is there a decentralized migration option for self-custody tokens?  
  **A**: Yes, through the migration portal that will be announced

- **Q**: Will Bybit support the migration?  
  **A**: Yes, Bybit supports migration for spot trading

- **Q**: Do we need gas fee for migration?  
  **A**: Yes, but very little

- **Q**: Does the migration support Phantom wallet?  
  **A**: Yes, Phantom will be supported

- **Q**: When will trading start for the new token?  
  **A**: Depends on the exchange; Binance should be trading 7 hours after migration

- **Q**: Will the value of tokens change after migration?  
  **A**: The value of your tokens shouldn't change

- **Q**: How long is the migration window open?  
  **A**: Window is open for 90 days

- **Q**: Will migrated tokens be tradeable right away?  
  **A**: Yes, that's right

### Development
- **Q**: Can we release the plugin-elizacloud plugin?  
  **A**: Yes, just bump the package.json version to publish it

- **Q**: Is there a problem with Sentry errors?  
  **A**: No, it's just one user spamming a CORS error from a weird browser, elizaos works fine

- **Q**: How should I deploy a Discord bot?  
  **A**: Use plugin-discord, and visit discord.com/developers/ for setting up keys
