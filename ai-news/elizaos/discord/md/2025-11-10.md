# elizaOS Discord - 2025-11-10

## Overall Discussion Highlights

### Token Migration Progress
- **Exchange Status**: Kucoin is actively working on the migration while Gate.io is awaiting feedback but expected to complete "this week"
- **Withdrawal Status**: Gate.io has paused withdrawals, indicating migration is underway
- **Team Confirmation**: Borko and Kenk confirmed that Gate.io has received all necessary information for the swap
- **Exchange Timeline**: Most exchanges are expected to address the migration this week
- **Non-Supporting Exchanges**: Users on platforms like Kraken Pro may need to withdraw to EOA wallets and follow migration portal steps

### Tokenomics Concerns
- **Supply Increase**: Community members expressed concern about the increased token supply (from 1B to 11B)
- **Team Clarification**: Only 13% is initially added to circulation, with team tokens locked for one year
- **Price Discussion**: Some users noted declining price despite broader market uptrend
- **Long-term Outlook**: Others remain optimistic, pointing to active GitHub development

### Product Development
- **ElizaCloud Timeline**: Borko confirmed release by year-end
- **Babylon Release**: Coming "much sooner" than ElizaCloud
- **Agent Game**: Team focused on developing an "agent game" for upcoming Devconnect event
- **OTC Agent**: Development team working on deploying an OTC agent, requiring PostgreSQL database
- **Next.js Application**: Team using Vercel for hosting but experiencing build errors

### Community Initiatives
- **Content Creation**: Lopez working on new videos about the migration process
- **Competition Ideas**: DorianD suggested implementing competitions for ElizaCloud with rewards for agents generating highest payments or reputation scores
- **Youth Entrepreneurship**: Discussion about creating opportunities for young entrepreneurs to generate income through AI agents by summer 2026
