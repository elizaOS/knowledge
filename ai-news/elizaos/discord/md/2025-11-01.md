# elizaOS Discord - 2025-11-01

## Overall Discussion Highlights

### Token Migration to ElizaOS
- Migration of AI16Z to ElizaOS begins November 7th with a 90-day window for token holders
- Supply will increase from 6.6 billion to 11 billion tokens (+40%)
- Circulating supply will immediately increase from 6.6 billion to 7.4 billion (13% increase)
- Tokenomics breakdown: 75% to community (60% swapped from original AI16Z holders), 15% to SAFT, 10% to team and contributors
- Migration will require using a portal that will be announced in official channels
- Exchanges have been notified but may not all support automatic migration

### Development Updates
- Eliza cloud release is being prepared for demonstration at Devconnect
- Team is developing an x402 gateway that integrates with DegenAI
- Ongoing research for on-chain integration of the cloud in collaboration with Ethereum Foundation around ERC-8004
- x402.elizaos.ai will serve as a central hub for APIs rather than listing partner agents
- Otaku Farcaster miniapp is in development, dependent on Coinbase integration, expected to launch next week
- Some x402 resource servers contain malicious code that needs fixing

### Technical Implementations
- Discussion about suppressing Bootstrap on custom slash commands in ElizaOS
- Current implementation uses a patch-based approach to prevent LLM from responding to specific commands (/f, /fl, /fc)
- Implementation involves patching the Telegram plugin to check for a custom flag

### Cryptocurrency Assets
- Conversation about Rare Pepes and Fake Rares on the counterparty platform
- Discussion about creating assets like PEPEPIZZA and potential subassets
