## Key Questions & Answers

**Q: Is Babylon launching as a web browser game or requiring local installation?**  
A: Web browser game (answered by Odilitime)

**Q: Are there plans to support migration for Bithumb users who held tokens before the snapshot?**  
A: We're working on a solution with them; if you had tokens before November 11th you'll be fine (answered by jasyn_bjorn)

**Q: How can I migrate my AI16Z tokens from SafePal wallet?**  
A: Just import your Safepal into phantom or any Solana wallet and migrate (answered by Hexx üåê)

**Q: What would be the correct way to declare the deepsek key in my.env file so that it can be detected?**  
A: OpenRouter provides DeepSeek as a model; you need an OpenRouter key, not a DeepSeek key (answered by sayonara)

**Q: I don't have tokens in OpenRouter, but I do have tokens in DeepSeek, is it possible to connect?**  
A: If you have a DeepSeek key, you can use that with the OpenAI plugin by replacing the endpoint URL. Check here: https://github.com/elizaos-plugins/plugin-openai (answered by sayonara)

**Q: Or is it that when you do the migrations, the tables are already created by default?**  
A: Yes (answered by sayonara)

**Q: Did Babylon repo move?**  
A: Yes, it's now at: https://github.com/BabylonSocial/babylon (answered by Odilitime)
