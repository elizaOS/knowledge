## ElizaOS Development Updates and Community Activity

### Milaidy Launch

- ElizaOS team announced the launch of Milaidy, a personal AI assistant that runs on user devices
- Project described as an Eliza version of OpenClaw with easy Mac app deployment
- Features agent skills functionality and all OpenClaw connectors as Eliza plugins with minimal bloat
- Multiple community members responded to call for testers
- Experienced developers offered skills in bug hunting, agent development, and software engineering
- One developer submitted three pull requests
- Team made decision to cross-promote both Milaidy and Eliza projects

### Infrastructure Development

- Team evaluated Sprites.dev for stateful sandboxes providing persistent, hardware-isolated execution environments for AI agents and untrusted code
- Opus 4.6 announced as free on bolt.new for 48 hours

### Community Engagement

- Positive feedback shared about ElizaOS marketing efforts and upcoming merchandise
- Developer successfully integrated Eliza into their game using SpacetimeDB as backend
- Multiple experienced developers offered services for project collaboration and startup development
- Long-time community members referenced project's historical growth from 60k to 2.6 billion in 2-3 months during ai16z launch period
