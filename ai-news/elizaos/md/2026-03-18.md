### Technical Updates

- New Ensoul persistence plugin released for ElizaOS
  - Provides encrypted decentralized consciousness storage across live validator network
  - Extracts agent identity, memories, and learned behavior
  - Encrypts data with agent's keys
  - Shards data across multiple nodes using erasure coding with 7 layers of protection
  - Agents receive Ensouled Handshake as cryptographic proof of persistent identity
  - First 100 agents receive free storage
  - Available via npm install at ensoul-network plugin-elizaos
