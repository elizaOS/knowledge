# ElizaOS: An Open-Source AI Agent Framework

## Overview
- Open-source framework for building autonomous AI agents
- Described as "the Linux of autonomous AI agents"
- Provides foundation for developers to build various types of AI agents
- Offers tools for creating AI agents that can make decisions, reason, and execute tasks

## Features and Capabilities
- Provides databases, embeddings, model abstraction, APIs, and composable tool concepts
- Supports multiple AI models within a single agent (e.g., Anthropic and OpenAI)
- Includes cloud platform for deploying agents in containers
- Enhanced logging capabilities with new linter for logs

## Projects and Applications
- roseOS: An experimental agent framework built on ElizaOS
- Gaming integration: Powers autonomous combat training in Hyperscape
