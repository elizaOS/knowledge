### The Undesirables Project

- A collection of 4,444 NFTs on Ethereum where each NFT's visual traits deterministically map to a unique personality across 7 dimensions
- Personality model based on the Big Five personality framework plus trading scores
- Each NFT workspace ships with ElizaOS V1 spec compatibility including YAML frontmatter and character.json export
- Features a 3-tier MemGPT-style memory system
- Includes a live market data pipeline injecting real-time data from CoinGecko, DeFiLlama, and DexScreener into agent sessions
- Developer invited feedback from others building in the space
