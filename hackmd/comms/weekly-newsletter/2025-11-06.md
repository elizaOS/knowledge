# ElizaOS Weekly Newsletter
**November 1-7, 2025**

## Executive Summary

This has been a landmark week for the ElizaOS ecosystem with several major milestones:
- The highly anticipated token migration from AI16z to ElizaOS is officially launching on November 7th, with a 1:6 conversion ratio
- Core release v1.6.4 was released with critical bug fixes and performance enhancements
- The Museum of Crypto Art (MOCA) project has launched 10,000 ElizaOS agents, marking a major ecosystem adoption milestone

## Development Updates

The development team has been highly active this week, with significant improvements to the core platform:

### Core Runtime & Security
- Fixed a critical bug in entity names array serialization for PostgreSQL databases, ensuring proper data handling across different input types
- Added a `skipMigrations` option to `runtime.initialize()`, allowing services to start without waiting for database migrations
- Identified and addressed a security concern with the `api/system/env/local` endpoint being accessible in development mode
- Enhanced the plugin-thedesk module for the OTC agent, resolving import issues

### New Features Under Development
Several ambitious features are currently in development:
- Entity-level row-level security is being implemented to provide more granular control over data access
- A dynamic prompt execution system is in progress to help prevent context length issues with smaller models
- Work continues on the X402 payment middleware for protecting plugin routes (EVM/Solana)
- A new `@elizaos/react` package with headless React hooks is being developed to enable external developers to build custom UIs

## Token Economics

The AI16z to ElizaOS token migration officially launches on November 7th with several key points:

- **Conversion Ratio**: 1 AI16z token will convert to 6 ElizaOS tokens
- **Migration Period**: Users will have a 90-day window to complete manual migrations
- **Exchange Support**: Multiple exchanges have confirmed support for the migration including MEXC, Bitget, Crypto.com, and Gate.io
- **Migration Portal**: A dedicated portal will be available starting November 7th for non-custodial wallet holders

Value accrual mechanisms for the ElizaOS token include:
- Eliza cloud revenue for buybacks, LP depth, and contributor rewards
- OTC agent desk revenue
- X402 payment protocol fees
- ERC-8004 product integrations
- Burns and fee integration as the AI agent economy grows
