## Cross-Issue Patterns / Themes
- **Auth/tenant/env drift repeatedly breaks onboarding**: staging vs prod origin/tenant mismatches, PKCE expectation changes, invite route mismatches, and signing/HMAC requirements create fragile seams.
- **Infra automation maturity is the gating factor for Cloud Apps**: Terraform state drift and manual SSH steps (“arming daemon”) are recurring operational hazards.
- **Scale-readiness requires explicit multi-node design**: single-node assumptions (placement, DNS, ingress) show up as latent launch blockers once concurrency increases.

   - signup success (non-invite path, if intended)
   - OAuth authorize + exchange success
   - email magic-link send + verify success  
   Alert on first failure.
