# Fact Briefing: 2026-03-28

## Overall Summary
The ElizaOS ecosystem is focusing on transitioning from a framework to a SaaS deployment platform (elizacloud) while integrating advanced autonomous trading and on-chain provenance features. High-velocity community development is centered on plugin registry submissions and clarifying the project's complex organizational boundaries.

## Key Facts

- Moderator Odilitime clarified Hyperscape is an incubated project or independent project by Shaw, not an official Eliza Labs product, on March 27.
- The xProof.app plugin was announced, providing on-chain decision provenance for agents on the MultiversX blockchain.
- ElizaOS operates on a model separating the open-source framework from a commercial SaaS platform called elizacloud for agent deployment.
- A developer submitted a trading flow plugin designed for plug-and-play integration between Eliza agents and trading infrastructure on March 27.
- The official ecosystem tokens are restricted to ElizaOS (Jeju) and DegenAI, both supported by the labs.
- Developer 'sailorpepe.eth' showcased an NFT-based system for 4,444 agents using a 3-tier MemGPT-style memory architecture and live market data feeds.
- Infrastructure consolidation is underway to collapse production and development environments into a single Spartan instance for customer data management.
- BitDelta exchange completed due diligence for AI16Z and is seeking marketing leads for a listing rollout plan.
