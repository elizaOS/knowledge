# Fact Briefing: 2026-04-01

## Overall Summary
The ElizaOS ecosystem is facing a critical trust crisis due to a 99.5% token price decline and allegations of team dumping, contrasted by a high-velocity technical phase focused on 'Agent Commerce' and security patching. A major supply chain vulnerability in the Axios package was identified and mitigated across several plugins.

## Key Facts

- A supply chain attack was identified in Axios version 1.14.1, which pulls in the malware package plain-crypto-js version 4.2.1.
- Developers pinned the Axios dependency to version 1.7.8 in plugin-autocoder and plugin-coingecko to mitigate the security risk.
- ElizaOS token holders reported a 99.5% price decline, characterized by some community members as a 'death spiral' caused by paying developers in tokens.
- Community members alleged that 40% of community tokens were transferred to team wallets during the token migration process.
- The Orbis API marketplace reached 300 listed APIs and 13 active paid subscriptions, implementing an HTTP 402 payment protocol for autonomous agents.
- A new plugin proposal for wallet reputation scoring and DeFi TVL verification was submitted to the elizaos-plugins registry.
- The SAID Protocol was introduced to provide agents with cryptographic digital passports on the Solana blockchain.
- GitHub activity remained active with a focus on a major database refactor (PR #6509) and parallel action execution (PR #6654).
