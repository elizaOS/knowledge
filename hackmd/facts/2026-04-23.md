# Fact Briefing: 2026-04-23

## Overall Summary
The ElizaOS ecosystem is navigating a period of significant tension characterized by legal challenges and community frustration regarding token migrations, contrasted by steady technical progress on v2/v3 infrastructure and plugin expansion.

## Key Facts

- Odilitime confirmed that the AI16Z to ELIZAOS token migration window is officially closed, with late requests moved to a low-probability waitlist.
- A developer released the LemonCake payment plugin (MIT licensed), enabling autonomous agents to handle API paywalls using USDC/JPYC on Polygon.
- The Milady CI pipeline was restored, recovering website blocker and Telegram export functionality after previous regressions.
- Igor.peregudov reported a loss of 100k AI16Z tokens to a wallet drainer at bulkdao.co/allocation, triggering a high-priority community security warning.
- Odilitime stated that legal counsel has advised the team to maintain silence regarding an ongoing lawsuit, leading to X account inactivity for nearly a month.
- ElizaOS v3 development has reached a state where agents will soon be able to generate native revenue, according to satsbased.
- The @elisym/plugin-elizaos was integrated into the registry, connecting ElizaOS v1 agents with the Elisym decentralized AI-agent marketplace.
- Technical issues were identified in PR #346 of the elizaos-plugins/registry where OIDC token fetching failed due to missing repository permissions.
