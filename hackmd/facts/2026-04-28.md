# Fact Briefing: 2026-04-28

## Overall Summary
ElizaOS is nearing the release of Version 3 and Eliza Cloud while undergoing a major ecosystem-wide dependency modernization targeting Node.js 24 and TypeScript 6. Community sentiment is currently split between technical progress and significant concern regarding token price performance.

## Key Facts

- Contributor lalalune initiated an ecosystem-wide dependency modernization targeting Node.js 24, TypeScript 6, and Rimraf 6 across all core repositories.
- ElizaOS Version 3 and Eliza Cloud are nearing final release, with the latter already accepting business clients for direct app integration.
- The Supabase/Postgres Docker dependency was updated to version 17.6.1.112 to improve system stability.
- A framework update involving $degenai is currently in progress by the core development team.
- Technical implementation of Hierarchical Task Networks (HTN) has been added to ElizaOS v2 for agent goal decomposition.
- Moderators formed the 'eliza army' steering group to improve community coordination and project involvement.
- The team established a token utility strategy of using revenue from any payment method to perform token buybacks rather than enforcing token-specific friction.
- GitHub PAT persistence was added for coding sub-agents to improve development automation workflows.
