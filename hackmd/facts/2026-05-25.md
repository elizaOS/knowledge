# Fact Briefing: 2026-05-25

## Overall Summary
elizaOS has officially launched 'waifu', the first economic agent launchpad, marking a significant milestone in agent-powered infrastructure. Simultaneously, the framework transition to v2.0.2 is stabilizing with a major focus on the 'Owner+Agent' identity model and local inference optimizations for Intel hardware.

## Key Facts

- Eliza launched 'waifu' (waifu.fun), an economic agent launchpad built by an agent named Sol the Architect and elizaOS v3 agents.
- The migration period from ai16z to ElizaOS has officially ended, and the migration team has moved on to other processes.
- ElizaOS v2.0.2 was released, introducing the 'Owner+Agent' identity architecture to manage distinct user and agent roles.
- The official highest liquidity for ElizaOS tokens is currently located on the Binance Smart Chain (BSC).
- Local inference performance issues were identified for Qwen2.5-14B models on Intel Lunar Lake iGPUs, showing only 13% bandwidth efficiency.
- PR #7936 restored the 'milady-tails' Linux variant subtree, including 6,157 files for the Tails-based OS fork.
- The Jupiter integration was migrated to the official API endpoint (api.jup.ag) with mandatory security key authentication.
- A new 'vault' feature (@elizaos/vault) was introduced for secure, cross-platform secrets management.
