# Fact Briefing: 2026-06-21

## Overall Summary
ElizaOS reached a major milestone with the transition of Eliza Cloud to production, though the launch has been accompanied by community fragmentation concerns regarding the waifu.fun token strategy and unresolved financial allegations against the Eliza Foundation.

## Key Facts

- Eliza Cloud platform successfully moved from staging to production, verified by a 10-user launch tracker in issue #8434.
- Development team resolved a critical 'too many clients' connection pooling error by transitioning managed agents to local in-container PGlite databases.
- Community members expressed concern over waifu.fun using a separate token instead of $ELIZAOS, citing potential market dilution.
- A major refactor split the monolith 'plugin-lifeops' into six specialized plugins and renamed the core to 'plugin-personal-assistant'.
- Users reported a protocol crash in the original $eliza token of 87% following the release of a new $ELIZA token.
- Allegations were posted regarding circular fund flows totaling 750k CAD between Eliza Foundation and Secure Blockchain Dev Corp.
- Eliza Cloud infrastructure was upgraded to a Cloudflare Worker model for the network gateway to improve efficiency.
- Mobile builds for Android and iOS were fixed to resolve long-standing parity issues between desktop and mobile features.
