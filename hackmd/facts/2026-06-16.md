# Fact Briefing: 2026-06-16

## Overall Summary
Development has hit a major milestone with the consolidation of all work into the primary Eliza repository and the final staging-to-production cutover for the Eliza Cloud Apps platform. Despite technical progress, the project faces persistent UI bugs on Windows and serious community allegations regarding financial transparency and circular fund flows.

## Key Facts

- Odilitime confirmed that all development work has been consolidated into the single eliza repository as of June 15.
- The Eliza Cloud Apps platform is in the final stages of a staging-to-production cutover with core features verified.
- NPM_TOKEN permissions for @elizaos/* packages and the 10-user launch tracking issue are currently halting the release pipeline.
- Intuition and MetaMask announced a $7,500 USDC bounty program for the ERC-7710 Delegation Framework and Knowledge Graph.
- The official ai16z token migration window has closed, and the community identified a fraudulent migration link targeting users.
- ElizaOS version 2.x development is occurring on the 'develop' branch while the 'main' branch remains on version 1.7.2.
- The Babylon platform status was confirmed as 'on hold' with social media components pending discussion with project leads.
- PR #8277 successfully decomposed the 'plugin-lifeops' monolith into six specialized, modular plugins to improve flexibility.
