# Fact Briefing: 2026-05-07

## Overall Summary
elizaOS has expanded its payment infrastructure by adding default payment methods for x402 routes and completed a major repository consolidation, merging ElizaCloud into the main eliza monorepo.

## Key Facts

- elizaOS and DegenAI were added as default payment methods in elizaOS for x402 routes as of May 6.
- ElizaCloud has been consolidated into the main eliza repository at github.com/elizaOS/eliza, migrating away from Vercel to container solutions.
- Shaw clarified the project operates as a fair launch without VC backing or a traditional formal team structure.
- The eliza-1 model training effort has transitioned to using Qwen as the base model with dflash speculative decoding and turboquant optimizations.
- Jupiter integration was migrated to the official api.jup.ag endpoint with mandatory API key authentication.
- The Slack plugin (@elizaos/plugin-slack) was migrated into the monorepo from its standalone repository.
- PR #7197 introduced @elizaos/vault, a cross-platform secrets vault with AES-256-GCM encryption at rest.
- Headless Linux segfaults (Issue #7231) and Telegram polling race conditions (Issue #7245) were resolved to improve system uptime.
