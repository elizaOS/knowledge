# Fact Briefing: 2026-01-11

## Overall Summary
The ecosystem is currently transitioning toward Eliza 2.0 and the Jeju Layer 2 launch, focusing on multi-language support and LLM-free agent architectures. While technical progress in gaming automation and cloud infrastructure is robust, the project faces significant headwinds from Korean exchange delistings and community debate over token utility.

## Key Facts

- Korbit exchange announced trading termination for ai16z and elizaOS tokens effective February 12, 2026.
- Jeju Layer 2 launch is targeted for H2 2026 with gas fees denominated in $elizaOS tokens.
- Eliza 2.0 is proposed as a multi-language runtime supporting TypeScript, Rust, and Python with FFI plugin interoperability.
- Shaw demonstrated LLM-free agents running a Game of Life simulation with 40 simultaneous agents using in-memory state parsing.
- The Hyperscape project implemented MMO agents for RuneScape and Roblox with anti-detection pixel-encoding for World of Warcraft.
- PR #6355 was introduced to add the plugin-blockrun for x402 micropayments on the Base chain.
- ElizaCloud enabled weather data API integration and processed its first cloud payment transactions.
- GitHub development activity on January 10 was exclusively concentrated on the elizaos.github.io repository.
