# Fact Briefing: 2026-03-20

## Overall Summary
The ElizaOS ecosystem is facing a severe liquidity and confidence crisis as the token reached new all-time lows, while technical development continues with new plugin releases and cloud infrastructure testing. Tensions are high regarding leadership absence and the perceived failure of the Milady-to-ElizaOS migration.

## Key Facts

- The ElizaOS token fell 99% from its previous high, dropping below the $10 mark into the $9 range on March 19, 2026.
- A new permissionless on-chain raffle plugin for the Base blockchain, 'moltraffle', was released with five core actions including Chainlink VRF integration.
- Eliza Cloud deployment encountered a critical 'Cannot find module @elizaos/plugin-discord' error during CLI-based AWS ECS deployment.
- Project leadership announced a shift in user feedback collection frequency from quarterly to weekly to accelerate development pace.
- The Ensoul persistence plugin (@ensoul-network/plugin-elizaos) was introduced to provide encrypted decentralized storage for agent consciousness.
- Framework refactoring in the xfn-framework standardized 'plugin-form-chain' and singularized 'plugin-form' naming conventions.
- The Milady app launch was scheduled for the evening of March 18, with the Babylon launch projected for early the following week.
- Eliza Cloud infrastructure specifications revealed a billing model of $1.17 per day with a 25-container maximum quota.
