# Fact Briefing: 2026-03-27

## Overall Summary
ElizaOS is transitioning from an open-source framework into a commercial SaaS ecosystem with the introduction of elizacloud, Jeju blockchain, and the xProof decision provenance plugin. Significant infrastructure consolidation is underway to streamline the DegenAI hosting environment into a single Spartan instance.

## Key Facts

- Odilitime confirmed ElizaOS and DegenAI are the only two tokens officially minted and supported by the labs.
- The xProof.app plugin was released to anchor agent decisions on the MultiversX blockchain for immutable provenance.
- Infrastructure consolidation is shifting production and development environments into a single staging Spartan instance.
- The AI16Z token (Jeju) is designated to power an upcoming native blockchain rather than serving as a memecoin.
- A partnership was announced with Nosana for a Builders' Challenge starting March 25 with workshops on March 26 and April 2.
- The Pythia MCP server was launched allowing agents to query Chainlink oracles on Polygon without API keys.
- BitDelta exchange offered a free listing for AI16Z following internal due diligence.
- A supply chain attack on the litellm-pypi package was identified as a security risk for the community.
