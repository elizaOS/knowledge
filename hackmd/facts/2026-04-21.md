# Fact Briefing: 2026-04-21

## Overall Summary
The ElizaOS ecosystem is transitioning towards autonomous revenue-generating agents with the announcement of v3, while managing security risks following a series of community phishing attacks and token migration closures.

## Key Facts

- The ai16z token migration period has officially closed as of April 20, 2026.
- ElizaOS v3 is nearing release and will feature capabilities for agents to generate revenue for users.
- Elisym Labs released @elisym/plugin-elizaos, enabling agents to operate as paid providers on a decentralized marketplace using Nostr protocols.
- ElizaOK was clarified as being built on ElizaCloud with no current separate token.
- Security vulnerabilities in the open-source application were privately disclosed to maintainers by researcher kullai on April 18.
- A memory management bug in InMemoryDatabaseAdapter.updateMemories was resolved to fix roomId handling.
- The core framework implemented a shared utils/batch-queue system to optimize high-concurrency tasks.
- The project team redirected social media attention to Milady development as a current priority over Twitter activity.
