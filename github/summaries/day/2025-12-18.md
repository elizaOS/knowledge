# elizaos/eliza Daily Update (Dec 18, 2025)
## OVERVIEW 
Today's development focused on significantly enhancing streaming support for text generation across various ElizaOS components, alongside improvements to the CLI test runner.

## KEY TECHNICAL DEVELOPMENTS

### Enhanced Streaming Capabilities for Text Generation
A major feature was introduced to enhance streaming support for text generation models. This update provides an opt-in streaming API via a `stream: true` parameter, ensuring backward compatibility while enabling more dynamic and responsive text generation. This involved extensive modifications across core, server, client, and API client packages. ([#6212](https://github.com/elizaos/eliza/pull/6212))

### CLI Test Runner Improvements
Improvements were made to the CLI test runner, specifically within the e2e-tests actions and general test runner utilities, contributing to a more robust testing environment. ([#6212](https://github.com/elizaos/eliza/pull/6212))
