# elizaos/eliza Daily Update (Dec 18, 2025)

## OVERVIEW 
Today's development focused on significantly enhancing streaming support for text generation models across the ElizaOS framework, alongside improvements to the CLI test runner. This involved extensive changes across core, server, client, and API client packages, ensuring a more responsive and efficient user experience.

## KEY TECHNICAL DEVELOPMENTS

### Enhanced Streaming Capabilities for Text Generation
A major feature was introduced to enhance streaming support for text generation models. This update provides an opt-in streaming API via a `stream: true` parameter, improving the responsiveness of text generation within the ElizaOS framework. The changes span across various packages, including `core`, `server`, `client`, and `api-client`, ensuring comprehensive integration of the new streaming functionality. ([#6212](https://github.com/elizaos/eliza/pull/6212))

### CLI Test Runner Improvements
Alongside the streaming enhancements, improvements were made to the CLI test runner. These updates contribute to a more robust and efficient testing environment for ElizaOS, particularly in the `packages/cli` directory. ([#6212](https://github.com/elizaos/eliza/pull/6212))
