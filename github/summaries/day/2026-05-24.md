**Agent Core and Safety Improvements**
*   Resolved planner regressions by binding runaway tool-argument validation loops and restoring evaluator message precedence ([elizaos/eliza#7924](https://github.com/elizaos/eliza/pull/7924), [elizaos/eliza#7897](https://github.com/elizaos/eliza/pull/7897)).
