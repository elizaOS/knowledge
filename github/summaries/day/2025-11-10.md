# elizaos/eliza Daily Update (Nov 10, 2025)
## OVERVIEW 
Today's development focused on addressing several bug fixes and opening new pull requests to improve environment variable loading and RLS validation.

## KEY TECHNICAL DEVELOPMENTS

### Environment Variable Handling
- A new pull request was opened to fix how environment variables are loaded, switching from `.env` files to `process.env` to ensure proper configuration. ([#6141](https://github.com/elizaos/eliza/pull/6141))

### Row-Level Security (RLS) Enhancements
- A fix was proposed for an issue where RLS `server_id` validation checks were blocking all users when RLS isolation was disabled. ([#6139](https://github.com/elizaos/eliza/pull/6139))

### API Stability Improvements
- A chore pull request was opened to remove a message emit in the `src` API to prevent race conditions. ([#6137](https://github.com/elizaos/eliza/pull/6137))
