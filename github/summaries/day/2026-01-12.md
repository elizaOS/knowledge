### Performance Enhancements & Core Features
*   Optimized `runtime.initialize()` in `elizaos/eliza` by implementing atomic upserts and parallelizing independent operations, reducing cold start times by 30% and warm start times by 40%. This involved changes across `plugin-sql`, `server`, and `core` packages. [elizaos/eliza#6342](https://github.com/elizaos/eliza/pull/6342)
*   Added support in `elizaos/eliza` for configuring the embedding dimension via the `EMBEDDING_DIMENSION` setting, eliminating a ~500ms embedding API call during agent initialization. [elizaos/eliza#6357](https://github.com/elizaos/eliza/pull/6357)

### Documentation & Dependency Updates
