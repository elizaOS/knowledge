**Eliza Cloud** is the managed backend for Eliza agents and apps. It runs on Cloudflare Workers and exposes a categorized HTTP API (50+ endpoint families) plus a React dashboard.

Cloud is **optional**. Agents and apps run fully local-only without it. Cloud is what you use when you want managed auth, hosted inference, billing, custom domains, container deploys, or multi-tenant isolation.

## What Cloud handles

| Surface       | What it gives you                                                              |
| ------------- | ------------------------------------------------------------------------------ |
| **Auth**      | OAuth (Google, Discord, X), API keys, Steward JWT.                             |
| **Apps**      | `appId` registration, versioning, publishing.                                  |
| **Inference** | Routed model calls with markup, rate limits, fallback chains.                  |
| **Containers**| Hosted agent container deploys with logs and lifecycle.                        |
| **Billing**   | Credits, top-up, Stripe, auto-renewal, creator earnings.                       |
| **Domains**   | Bind custom domains to deployed apps and agents.                               |
| **MCP / A2A** | Hosted Model Context Protocol and agent-to-agent endpoints.                    |
